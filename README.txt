WHAT IS QLIK-AUTH-EXAMPLE?

  qlik-auth-example is a minimal example to demonstrate how the qlik-auth
  module works and a simple way of getting started working with it. See the
  documentation for qlik-auth for further details and instructions.

  https://github.com/braathen/qlik-auth

REQUIREMENTS

 - Node.js (including npm) <https://nodejs.org>

INSTALLATION

    npm install qlik-auth-example

RUNNING
	node app.js
